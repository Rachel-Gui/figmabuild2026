# CogniSense

CogniSense is an adaptive learning interface prototype for AI-assisted education. It was designed as a cleaner study workspace that combines lesson planning, guided knowledge boards, saved study folders, learning analytics, and profile tracking in a single frontend experience.

## Overview

This project focuses on interface design and learning-flow organization rather than backend integration. The current version includes:

- A structured workspace with live conversation, progress tracking, and an independent knowledge board
- Interactive knowledge tabs with add/remove behavior
- A folder view for saved study records and previous learning sessions
- A dashboard with learning metrics and visualization panels
- A profile page for goals, streaks, highlights, and achievements

## Local Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Notes

- This is currently a frontend-only prototype.
- No backend, database, cloud service, or external API is connected yet.
- The original design source came from a Figma-generated project scaffold and was then iteratively redesigned in code.
